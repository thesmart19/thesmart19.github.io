В нашей лицензии есть доступ к форматам:
Mobile - мобильная реклама Android / iOS
Desktop - реклама на устройствах Mac / Windows
Native - нативная реклама на Mac / Windows / Android / iOS
Push - реклама формата Push Notifications

Adplexity Mobile

https://mobile.adplexity.com/

Основные блоки интерфейса: меню поиска и фильтры.

Поиск:

  1. Поиск "По Запросу" позволяет найти креативы по названию оффера или ключевым словам в тексте креатива
  2. "По рекламодателю" - поиск креативов по ссылке на оффер или лендинг.
  3. "По площадке" - поиск по названию сайта, на котором размещена реклама

В левой части страницы расположены Фильтры поиска.

Фильтры поиска:

  1. Даты - период времени
  2. Активно (дней) - сколько дней креатив был на трафике
  3. Тип рекламы - popups, mobile web and in-app
  4. Тип устройства - Android / iOS
  5. Размер - размер баннера/креатива
  6. Страна - Geo таргегинг рекламы
  7. Язык - позволяет отфильтровать креативы по языку
  8. Подключение - таргетинг по типу интернет соединения
  9. Источник - рекламные сети
  10. Партнерская сеть - CPA партнерки
  11. Трекер - фильтр по используемому трекеру
  12. Цена bid - ставка
  13. Выбор языка интерфейса Adplexity

Case: LeoVegas

Кейс: поиск креативов для офера казино LeoVegas
Geo: Brazil
Platform: Android
Переходим на страницу https://mobile.adplexity.com/
Выбираем поиск "По запросу". В поле "Запрос" вводим название офера. Можно выбрать сортировку креативов: "Новые", "Давно запущенные" или "С большим траффиком". Добавляем фильтр по Geo таргетингу (Brazil).
Получаем подборку креативов.

Пример поиска креатива "По рекламодателю".
Этот тип поиска позволяет найти креатив по ссылке Target URL оффера.
В поле "Поиск в" выбираем тип URL по которым будет производиться поиск. Копируем Target URL в соседнее поле. Выбираем тип сортировки, заполняем фильтры.
Получаем подборку креативов.

Клик по креативу открывает страницу с подробным описанием креатива.

Страница "Подробное описание креатива":

  1. Popup - скриншот креатива.
  2. Страна - geo таргетинг
  3. Подключения - таргетинг по типу интернет соединения
  4. Целевые страницы - клик по URL покажет полную ссылку.
  5. Тренд и Тренд за неделю - даты и количество показов креатива.
  6. Изменение Bid - даты, страны и величина ставки (Bid)
  7. Тип устройства - соотношение количества показов креатива на разных устройствах.
  8. Площадки по времени и по трафику - соотношение количества показов креатива на сайтах.
  9. Источники и коды размещений - соотношение показов креатива в рекламных сетях.

Download landing

Adplexity позволяет скачать креатив.
Для скачивания нужно перейти на страницу "Подробное описание креатива" и кликнуть на кнопку "Еще" в блоке Целевые страницы.

Клик по кнопке "Еще" открывает диалоговое окно "Детали целевой страницы".
В диалоговом окне показан список редиректов и ссылки на страницы оффера.
В нижней части страницы находится кнопка "Скачать целевую страницу".
Клик по кнопке запустит скачивание ZIP архива с исходным кодом креатива.

После скачивания ZIP архив нужно распаковать и подготовить лендинг к загрузке на хостинг.

Код каждого лендинга уникален, но можно выделить несколько общих шагов:

  1. Код лендинга чаще всего находится в файле *.html
  2. Файл *.html можно редактировать любым текстовым редактором.
  3. В файле *.html нужно найти ссылки на оффер и заменить на актуальные ссылки.
  4. Нужно найти ссылки на скрипты трекера, push-теги, системы web-аналитики и удалить или заменить на актуальные ссылки.

Ссылки на оффер и скрипты можно найти в коде по словам http и https
<a href="http://...">...</a>
<script src="https://...">...</script>
<script type="text/javascript">...</script>
var offerurl= "https://..."
